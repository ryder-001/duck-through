# 🚪 钻洞大冒险 (Duck Through)

一款适合小朋友玩的亲子互动小游戏。爸爸驮着孩子穿过家里的各个房间，遇到矮门洞时儿子要赶紧低头！

## 🎮 玩法

- **按住空格键 / 点击屏幕** — 儿子低头，帮助穿过矮门洞
- **松开** — 恢复站立
- 穿过门洞得分，收集金币加分，连续通过有连击奖励
- 撞到门框或吊灯会扣生命值，3条命用完游戏结束

## 🏠 场景

游戏包含5个不同的房间场景，每过6道门进入下一个房间：
- 客厅 → 卧室 → 厨房 → 书房 → 地下室

每个房间都有语音播报告诉小朋友到了哪里。

## ✨ 特性

- 正面视角卡通人物，爸爸驮着孩子的形象清晰可爱
- 真人中文语音播报（Tingting）
- 背景音乐 + 效果音
- 关卡递进难度系统
- 金币收集、连击奖励
- 适配手机和电脑
